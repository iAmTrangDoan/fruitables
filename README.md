# Đồ án Website

## 1. Yêu cầu môi trường
- PHP >= 8.0
- MySQL
- Apache (WAMP / XAMPP)

## 2. Cài đặt database
1. Tạo database: organic_store
2. Import file: database/organic_store.sql
3. Cấu hình kết nối tại: config/config.php
